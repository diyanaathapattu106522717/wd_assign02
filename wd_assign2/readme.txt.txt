JavaScript Validation:
Each form has a validation function that checks the user’s input before the form submits. 
The script makes sure required fields aren’t empty, emails and phone numbers look valid, 
a restaurant is selected, the date/time is in the future, and the correct payment details 
are entered depending on the deposit method. If something is wrong, an error message shows 
and the form doesn’t submit until everything is fixed.
